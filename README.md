# TaskFlow - Professional To-Do List Web Application

![TaskFlow Banner](screenshots/banner.png)

## 📌 Project Overview

TaskFlow is a modern and responsive To-Do List Web Application designed to help users manage daily tasks efficiently. The application provides a clean user interface with advanced productivity features including task management, filtering, searching, sorting, priority handling, due dates, dark mode, and progress tracking.

This project was developed using core web technologies and focuses on improving productivity through a simple yet powerful task management experience.

---

## 🚀 Features

### ✅ Task Management

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Delete all completed tasks

### 🔍 Search & Filtering

* Search tasks instantly
* Filter tasks by:

  * All Tasks
  * Pending Tasks
  * Completed Tasks

### 📊 Productivity Dashboard

* Total task counter
* Completed task counter
* Task completion percentage
* Dynamic progress bar

### 🔥 Task Organization

* Add task priority:

  * High
  * Medium
  * Low
* Add task due dates
* Sort tasks by:

  * Newest
  * Oldest
  * Priority
  * Due Date
  * Alphabetical order

### 🌙 User Experience

* Dark mode support
* Theme preference saved automatically
* Toast notifications
* Delete confirmation modal
* Responsive design for mobile and desktop devices

### 💾 Data Storage

* Uses browser Local Storage
* Tasks remain available after refreshing the page

---

## 🛠️ Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript (ES6)

### Storage

* Browser Local Storage

### Tools

* Visual Studio Code
* Git & GitHub
* Chrome Developer Tools

---

## ⚙️ Installation & Running the Project

### 1. Clone Repository

```bash
git clone YOUR_REPOSITORY_URL
```

### 2. Open Project Folder

```bash
cd TaskFlow
```

### 3. Run Application

Open:

```
index.html
```

in your preferred browser.

No server setup is required because the application uses browser Local Storage.
